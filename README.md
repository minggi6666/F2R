
This is the official PyTorch implementation of **Frames2Residual: Spatiotemporal Decoupling for Self-Supervised Video Denoising, accepted to ECCV 2026.

## Abstract

> Self-supervised video denoising methods typically extend image-based frameworks into the temporal dimension, yet they often struggle to integrate inter-frame temporal consistency with intra-frame spatial specificity. Existing Video Blind-Spot Networks (BSNs) require noise independence by masking the center pixel, this constraint prevents the use of spatial evidence for texture recovery, thereby severing spatiotemporal correlations and causing texture loss. To address this, we propose Frames2Residual (F2R), a spatiotemporal decoupling framework that explicitly divides self-supervised training into two distinct stages: blind temporal consistency modeling and non-blind spatial texture recovery. In Stage 1, a blind temporal estimator learns inter-frame consistency using a frame-wise blind strategy, producing a temporally consistent anchor. In Stage 2, a non-blind spatial refiner leverages this anchor to safely reintroduce the center frame and recover intra-frame high-frequency spatial residuals while preserving temporal stability. Extensive experiments demonstrate that our decoupling strategy allows F2R to outperform existing self-supervised methods on both sRGB and raw video benchmarks.

## Pretrained models

Please place all pretrained models and released checkpoints into the `models/` folder.

```text
models/
├── ISP_CNN.pth        # ISP network for RAW visualization/evaluation
├── nafnet_raw.pth     # Pre-trained RAW image denoiser
├── nafnet_rgb.pth     # Pre-trained sRGB image denoiser
├── raw.pth            # Released F2R RAW video denoising model
└── rgb_sigma30.pth    # Released F2R sRGB Gaussian sigma=30 model
```

## Quick test

### sRGB video denoising

Run the following command to evaluate the released sRGB F2R model:

```bash
python rgb_val.py \
  --checkpoint ./models/rgb_sigma30.pth \
  --nafnet_path ./models/nafnet_rgb.pth \
  --val_dir_davis ./datasets/DAVIS/test \
  --val_dir_set8 ./datasets/Set8/test \
  --noisetype gauss30 \
  --gpu_devices 0
```

### RAW video denoising

Run the following command to evaluate the released RAW F2R model:

```bash
python raw_val.py \
  --checkpoint ./models/raw.pth \
  --nafnet_path ./models/nafnet_raw.pth \
  --isp_path ./models/ISP_CNN.pth \
  --crvd_root ./datasets/CRVD \
  --test_iso 25600 \
  --gpu_devices 0
```

## Train

F2R is trained progressively with two stages. Stage 1 learns a blind temporal anchor from neighboring frames. Stage 2 refines the spatial residual by reintroducing the center frame.

### Stage 1: sRGB video denoising

```bash
python train_s1_rgb.py \
  --data_dir ./datasets/train \
  --val_dir_davis ./datasets/DAVIS/test \
  --val_dir_set8 ./datasets/Set8/test \
  --nafnet_path ./models/nafnet_rgb.pth \
  --noisetype gauss30 \
  --batch_size 4 \
  --len 9 \
  --gpu_devices 0
```

### Stage 2: sRGB video denoising

```bash
python train_s2_rgb.py \
  --data_dir ./datasets/train \
  --val_dir_davis ./datasets/DAVIS/test \
  --val_dir_set8 ./datasets/Set8/test \
  --nafnet_path ./models/nafnet_rgb.pth \
  --stage1_path ./models/rgb_stage1.pth \
  --noisetype gauss30 \
  --batch_size 4 \
  --len 9 \
  --gpu_devices 0
```

### Stage 1: RAW video denoising

```bash
python train_s1_raw.py \
  --data_dir ./datasets/CRVD \
  --nafnet_path ./models/nafnet_raw.pth \
  --batch_size 8 \
  --len 7 \
  --patch_size 256 \
  --gpu_devices 0
```

### Stage 2: RAW video denoising

```bash
python train_s2_raw.py \
  --data_dir ./datasets/CRVD \
  --nafnet_path ./models/nafnet_raw.pth \
  --stage1_path ./models/raw_stage1.pth \
  --batch_size 8 \
  --len 7 \
  --patch_size 256 \
  --gpu_devices 0
```

For RAW Stage 2 training, F2R does not require manually specifying a fixed noise type. The pseudo center frame generated by Stage 1 is randomly degraded with the RAW noise model during training, and then combined with neighboring noisy frames for residual refinement.

### Resume or warm-start Stage 2

```bash
python train_s2_raw.py \
  --data_dir ./datasets/CRVD \
  --nafnet_path ./models/nafnet_raw.pth \
  --stage1_path ./models/raw_stage1.pth \
  --stage2_init_path ./experiments/your_exp/models/epoch_Fusion_125.pth \
  --start_epoch 126 \
  --gpu_devices 0
```
